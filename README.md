# 毕业设计项目前端

## 项目简介

该项目是我的毕业设计项目(前端部分)，使用 React 技术栈进行开发。该项目旨在实现一个NLP情感分析系统。

## 前端技术栈

- React：用于构建用户界面的 JavaScript 库。
- React Router：用于实现前端路由的库。
- MobX-react：用于状态管理的库。
- Ant Design：用于构建用户界面的 UI 组件库。

- Node版本: 16.20.2
